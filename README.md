# Prework Study Guide Webpage

## Description

- The purpose of this project is to help build the foundations of a web developer 
- This project was created to help aspiring web developers learn the foundations of web development 
- This project helps new students get a basic grasp of web development 
- Creating this project helped me understand the importance of Git and Github. As well as furhting my understand of HTML, CSS and JavaScript

## Usage

This webpage includes basic information about HTML, CSS, Git, and JavaScript. It is divided into four sections so that the user can easily locate
information. If the user is not sure what to study, they can open up the console (on windows press ctrl + shift + I) to get a suggestion. The web 
page uses JavaScript to list the four subjects and provide a suggestion.
