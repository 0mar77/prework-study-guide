HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The ul element is for unordered list, while the ol element is for ordered list
  • The li (list items) element is used to list the items in the list
  • We can use the link element to create a connection with an external file

CSS

  • Inline CSS is applied to a specific HTML tag
  • Internal CSS are in the head section of an HTML file and the rules only apply to that page
  • External CSS sheets are kept on a separate file
  • CSS rules are composed of a selector and declarations
  • Selector defines the element or attributes to which the rules, or declarations, will apply
  • Declarations are made up of 2 components. The property and the value of the property
  • A period is used to designate class selectors
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • The box-shadow property adds shadow effects around an elements frame

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and swithces to it

JavaScript

  • A variable is a named container that allows us to store data in our code
  • An "if" statement is a way of introducing decision making into code. It will only execute if the statement meets a condition, or is true
  • The "else if" statement can be used if you want to have more than one conditional
  • Control flow is the order in which a computer executes code in a file. Top to bottom
  • A value is consdered falsy if it is a 0, -0, "", null, undefined or NaN. Everything else would be truthy
  • An "array" is a single variable thais is used to hold a group of data
  • An "index" is the location of the data item in the array
  • A "for" loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
  • Functions are reusable blocks of code that contain instructions to perform a specific task
